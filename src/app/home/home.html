What's this?

Appoints is an appointment scheduling sample application that hopefully provides a bit of guidance into the wild west that is called 'Modern Day Web Development'. It consists of a Node.js REST API backend and an AngularJS frontend, so it's totally buzzword-compliant. Apart from that, JavaScript, Node.js and Angular simply rock and make it very much possible to build clean, testable and maintainable apps.

The following design decisions have been taken:

  • Keep it as simple as possible
  • Totally separated client and server API, no state;
  • JWT Token based authentication/authorization;
  • No local user accounts, only 3rd party (Facebook & Google implemented);
  • REST API with a touch of hypermedia (HAL);
  • Client build step with Gulp, no messing with bundles, asset pipeline or requirejs.
